# v0.4.0 Release Assets

Place the following demo GIFs and images in this directory:

## Required Files

### Hero Image
- `hero.png` - Main hero image for the release (1200x800 recommended)

### Author
- `dylan-avatar.png` - Avatar for the floating video player (200x200)

### Demo GIFs
Record these using a screen capture tool like [Kap](https://getkap.co/) or [CleanShot X](https://cleanshot.com/):

1. **`workspace-demo.gif`** - Creating and managing workspaces
   - Show creating a new workspace
   - Inviting a team member
   - Switching between workspaces

2. **`tables-demo.gif`** - Creating and managing tables
   - Create a new table
   - Add fields (show the AI field generator if possible)
   - Add some sample data
   - Filter/sort the data

3. **`page-builder-demo.gif`** - Visual page builder
   - Create a new page
   - Add a List block connected to a table
   - Add a Record block
   - Show the preview

4. **`ai-chat-demo.gif`** - AI chat with mentions
   - Open the AI chat
   - Type @ and show the mention dropdown
   - Select a table or page
   - Send a message and show the AI response

## Tips for Recording GIFs

- Use 1280x800 or similar resolution for clarity
- Keep GIFs under 10 seconds each
- Use a clean, populated dataset for demos
- Record at 15-20 fps, optimize with [gifsicle](https://www.lcdf.org/gifsicle/) or similar
